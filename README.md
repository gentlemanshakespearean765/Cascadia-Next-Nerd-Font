## 🖥️ What This Is

Cascadia-Next-Nerd-Font is a patched version of Cascadia Next with full Nerd Fonts icons and three CJK variants:

- Simplified Chinese
- Traditional Chinese
- Japanese

It keeps the Latin and CJK width balance at 2:1, so text lines stay neat in terminals and editors. It works well for daily use when you want clear letters, clean spacing, and icon support in one font.

## 🔤 Spacing and Width

This font uses a 2:1 width balance between Latin and CJK text. That means:

- English text stays compact
- Chinese and Japanese text line up well
- Mixed text looks tidy in terminals
- Command output is easier to read
- Editor text feels more even across lines

This helps when you work with shell output, file names, logs, and text notes that mix languages.
